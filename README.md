## вңЁ Features

| Feature | Description |
|---------|-------------|
| рҹҺҙ **Interactive flashcards** | 3D flip, smooth navigation, and a dark theme optimized for long study sessions |
| рҹ§® **Adapted SM-2 algorithm** | Smart spaced repetition that optimizes when to review each card |
| вҢЁпёҸ **Keyboard shortcuts** | Study without touching your mouse: Space to flip, 1-4 to rate |
| рҹ“Ҡ **Visual statistics** | Progress charts, difficulty distribution, and mastery evolution |
| рҹ“қ **Card editor** | Create, edit, and delete cards inside any deck |
| рҹ—ӮпёҸ **Main decks + Extras** | Two core subjects always visible; the rest tucked inside a collapsible panel |
| рҹ’ҫ **Local persistence** | All data is saved to localStorage; works without a server |
| рҹ“ұ **Responsive** | Optimized for mobile, tablet, and desktop |

## рҹ“Ұ Local setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/anki-cards.git
cd anki-cards

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev

# 4. Open in your browser
# http://localhost:5173
```

### Production build

```bash
npm run build
```

Output is generated in the `dist/` folder, ready to deploy on Vercel, Netlify, GitHub Pages, or any static CDN.

## рҹ§  Adapted SM-2 algorithm

The spaced-repetition system implements an adapted version of the SM-2 algorithm:

| Difficulty | Ease factor change | Base interval |
|------------|--------------------|---------------|
| рҹ”ҙ Again   | вҲ’0.20              | 1 day         |
| рҹҹ  Hard    | вҲ’0.15              | 2 days        |
| рҹҹў Good    | No change          | 3 days        |
| рҹ”ө Easy    | +0.15              | 4 days        |

New cards progress through the states: `new вҶ’ learning вҶ’ review`.

## рҹ“қ Adding new subjects

1. Create a JSON file at `public/data/my-subject.json` following this schema:

```json
{
  "id": "my-subject",
  "name": "My Subject",
  "description": "Short description",
  "subject": "Category",
  "cards": [
    {
      "id": "uuid",
      "front": "Question",
      "back": "Answer",
      "tags": ["tag1"],
      "difficulty": "medium"
    }
  ]
}
```

2. Register it in `src/App.jsx` inside the `deckFiles` array.

3. Reload the application.
