La  forma  más  sencilla  de  alcanzar  este  objetivo  es  almacenar  la  información

La  principal  función  de  una  aplicación  informática  es  la  manipulación  y  la  transformación  
de  datos.  Estos  datos  pueden  representar  cosas  muy  diferentes  según  el  contexto  del  
programa:  notas  de  estudiantes,  una  recopilación  de  temperaturas,  las  fechas  de  un  
calendario,  etc.  Las  posibilidades  son  ilimitadas.  Todas  estas  tareas  de  manipulación  
y  transformación  se  llevan  a  cabo  normalmente  mediante  el  almacenamiento  de  los  
datos  en  variables,  dentro  de  la  memoria  del  ordenador,  de  modo  que  pueden  aplicarse  
operaciones,  ya  sea  mediante  operadores  o  la  invocación  de  métodos.

Desafortunadamente,  todas  estas  variables  sólo  tienen  vigencia  mientras  el  programa  
se  está  ejecutando.  Una  vez  finalizado  el  programa,  los  datos  que  contienen  
desaparecen.  Esto  no  es  problema  para  programas  que  siempre  tratan  los  mismos  
datos,  que  pueden  tomar  la  forma  de  literales  en  el  programa.  O  cuando  el  número  de

para  gestionarlas.  No  tiene  sentido.  Por  tanto,  en  algunos  casos,  aparece  la  necesidad

programas,  independientemente  de  su  sistema  operativo.

archivos,  en  el  apartado  “Tratamiento  básico  de  datos”  se  explica  cómo  poder  leer  y

1.  Gestión  de  archivos
Entre  las  funciones  de  un  sistema  operativo  está  la  de  ofrecer  mecanismos  genéricos  para  
gestionar  sistemas  de  archivos.  Normalmente,  dentro  de  un  sistema  operativo  moderno  (o  
ya  no  tan  moderno),  se  espera  disponer  de  algún  tipo  de  interfaz  o  explorador  para  poder  
gestionar  archivos,  ya  sea  gráficamente  o  usando  una  línea  de  comandos  de  texto.  Si  bien  
la  forma  en  que  los  datos  se  guardan  realmente  en  los  dispositivos  físicos  de  
almacenamiento  de  datos  puede  ser  muy  diferente  según  cada  tipo  (magnético,  óptico,  
etc.),  la  forma  de  gestionar  el  sistema  de  archivos  suele  ser  muy  similar  en  la  inmensa  
mayoría  de  los  casos:  una  estructura  jerárquica  con  carpetas  y  archivos.
1.1  La  clase  File
La  pieza  más  básica  para  poder  operar  con  archivos,  independientemente  de  su  tipo,  en  
un  programa  hecho  en  Java  es  el  tipo  compuesto  File.  Esta  clase  pertenece  al  package  
java.io.  Por  tanto,  deberá  importarlo  antes  de  poder  usarla.  Ésta  le  permite  manipular

En  este  sentido,  un  listado  de  10  puntuaciones  es  una  información  que  se  puede  tratar
