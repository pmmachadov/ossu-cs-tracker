Composición (◆ rombo relleno) vs Agregación (◇ rombo vacío)

classDiagram
    class Cientifico {
        +String nombre
        +String especialidad
        +investigar()
    }
    class Proyecto {
        +String nombre
        +String fechaInicio
        +ejecutar()
    }
    class Asignacion {
        +String rol
        +Date fechaInicio
    }
    class Especialidad {
        <<enumeration>>
        BIOLOGIA
        GEOLOGIA
        OCEANOGRAFIA
    }

    Proyecto o-- Cientifico : agregacion
    Proyecto *-- Asignacion : composicion
    Cientifico *-- Asignacion : composicion
    Cientifico --> Especialidad : usa
